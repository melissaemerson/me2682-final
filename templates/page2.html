Topics of Assignments for Spring 2020

  • Homework 1: Binary numbers, introduction to algorithms, truth tables, structures of programs, list operations
  • Homework 2: Selection sort, sequential search, time complexity, projectile motion simulation, utility functions for breast cancer dataset analysis
  • Homework 3: Set operations, time complexity, dictionaries, recursion, numpy, vertical percolation, utility functions for breast cancer dataset analysis
  • Homework 4: Object oriented programming of a course with students, professors, and assignments
  • Midterm: PPM image editor with functions that flip the image horizontally, turn it black and white, and filter unwanted objects appearing in different parts of an image
  • Homework 5: Machine learning program that gives statistical information, creates plots, analyzes a breast cancer dataset using knn algoritm, pandas, seaborn
  • Final: Create a webpage using Flask and add it into a Git repository
